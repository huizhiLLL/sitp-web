# FC-Catalyst Web

一个基于 VitePress 的燃料电池催化剂性能预测平台与文档中心。

## 简介

FC-Catalyst Web 是一个集成了机器学习预测功能的综合性平台，专注于燃料电池催化剂性能的预测与分析。项目采用 VitePress 构建，兼具预测工具与文档中心双重功能。

### 特点

- **性能预测平台**：基于机器学习模型，提供燃料电池催化剂性能预测功能
- **文档中心**：使用 VitePress 构建，提供完整的技术文档与使用指南
- **数据提取**：集成 YOLO 计算机视觉技术，实现科学文献数据的自动化提取
- **现代化界面**：基于 Vue 3 和 VitePress，提供响应式、组件化的用户界面


## 技术栈

- **VitePress** - 静态站点生成器
- **Vue 3** - 前端框架
